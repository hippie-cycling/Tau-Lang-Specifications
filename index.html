Compiling Tau Language (pre-release).

Check the Github Rpository

As of today (March 19, 2024) Tau language can only be compiled on Linux (in my case I am using Pop-Os).

~Start by cloning the tau-lang repository. Open the terminal and type:

git clone https://github.com/IDNI/tau-lang.git


~The tau-lang repository should now be in your files

~Navigate to the tau-lang directory. In the terminal type:

cd tau-lang

~Lets compile. In the terminal type:

./release.sh ~or ./debug.sh

~Depending on the build to compile. In my case I will compile the release.


~Compiling takes a while, specially if you are using a VM.

~Once compiled, navigate to build-Release or build-Debug (depending on the build) by typing in the terminal:

cd build-Release

~Launch the Tau REPL by typing:

./tau
